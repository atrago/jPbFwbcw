# 项目简介

## 核心功能
- 基于YOLOv8的水下垃圾检测系统，支持识别12类垃圾：bio, plastic, rov, timestamp, unknown, metal, wood, rubber, cloth, fishing, paper, papper
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时显示功能：目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 提供参数调整：自定义置信度、IOU阈值
- 支持选择显示标签和原图
- 可选择使用不同检测模型

## 适用场景
- 水下环境垃圾检测与分类
- 海洋环境保护监测
- 水下机器人(ROV)辅助检测
- 水域生态研究
- 环保数据收集与分析

## 技术特点
- 基于深度学习框架PyTorch 1.9
- 使用YOLOv8目标检测算法
- 集成PyQt5图形界面
- 包含7000张水下垃圾标注数据集
- 提供完整的源代码和训练好的模型
- 支持多种开发环境：PyCharm+Anaconda 或 VSCode+Anaconda
